# Panda Productions Portfolio

A modern, interactive portfolio website built with React and Vite, showcasing creative projects and professional work.

## Features

- Responsive design with Tailwind CSS
- Smooth animations using GSAP
- Interactive components with Radix UI
- PDF viewer integration
- Dark/Light theme support
- Optimized for performance with Vite

## Tech Stack

- **Frontend**: React 19, Vite
- **Styling**: Tailwind CSS
- **Animations**: GSAP
- **UI Components**: Radix UI, Headless UI
- **PDF Viewing**: React PDF Viewer
- **Deployment**: Vercel

## Getting Started

### Prerequisites

- Node.js (version 18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd panda-productions
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
